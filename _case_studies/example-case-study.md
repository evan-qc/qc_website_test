## The challenge

The client was juggling estimates in one app, contracts in email, and project
status in a spreadsheet — re-entering the same information three times and
losing track of jobs in between.

## What we built

We embedded with the team, mapped how the work actually flowed, and connected
their existing tools into a single automated pipeline so an approved estimate
generates the contract and opens a tracked project automatically.
